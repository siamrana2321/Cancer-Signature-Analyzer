# COSMIC Signature Analyzer

A bioinformatics platform for identifying COSMIC mutational signatures from cancer genome VCF files.

## Features

- VCF parsing
- SBS96 context extraction
- COSMIC v3.6 signature loading
- NNLS signature decomposition
- Reconstruction quality evaluation
- Spring Boot backend integration

## Tech Stack

### Python
- NumPy
- Pandas
- SciPy
- scikit-learn
- Matplotlib

### Java
- Spring Boot
- Maven

## Workflow

VCF → SBS96 Profile → COSMIC Signatures → NNLS → Biological Interpretation
